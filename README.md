# Pro Logica Small Business Cybersecurity Checklist

A clean, lightweight static checklist tool that helps small business owners review basic cybersecurity readiness.

The checklist covers passwords, two-factor authentication, email security, software updates, backups, device security, access control, network security, and incident response. It includes browser-saved progress, a readiness percentage, and a simple risk level.

Live Demo: https://morphius999.github.io/prologica-small-business-cybersecurity-checklist/

This free checklist was created by Pro Logica to help small business owners review basic cybersecurity readiness.

Built by Pro Logica: https://www.prologica.ai

Pro Logica provides custom software development, AI automation systems, and cybersecurity assessment tools for growing businesses.

## Why This Exists

Many small businesses know cybersecurity matters but do not have a simple starting point for reviewing day-to-day security hygiene. This project provides a practical, no-login checklist that can be opened directly in a browser or published with GitHub Pages.

## Setup

Open `index.html` in a browser.

No frameworks, npm packages, or external dependencies are required.
